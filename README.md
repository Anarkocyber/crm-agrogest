🌱 CRM AgroGest

Sistema CRM desenvolvido para gestão e captação de clientes no setor agrícola, com foco em organização do funil de atendimento e acompanhamento de solicitações em cada etapa do processo.

Projeto Integrador — UNIVESP | Tecnologia da Informação

Sobre o Projeto
 
O CRM AgroGest nasceu da necessidade de pequenos e médios produtores rurais terem uma ferramenta simples e visual para:

Registrar e acompanhar clientes
Controlar em qual etapa cada solicitação se encontra
Facilitar a captação e o relacionamento com novos clientes

Funcionalidades

📋 Cadastro de clientes
🔄 Controle de etapas do atendimento (pipeline de solicitações)
🎨 Interface intuitiva pensada para o usuário final
📊 Visão geral do status de cada cliente
